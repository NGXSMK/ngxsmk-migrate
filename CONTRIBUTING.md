# Contributing to ngxsmk-migrate

Welcome! We are excited that you want to contribute to the future of Angular modernization.

## How to Contribute

1. **RFC Proposals**: For major changes, please open a [GitHub Discussion](https://github.com/ngxsmk/ngxsmk-migrate/discussions) using the RFC template.
2. **Bug Reports**: Use the [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md) for any issues you find.
3. **Good First Issues**: If you are new, look for the `good-first-issue` label.
4. **Pull Requests**:
   - Fork the repo.
   - Create a feature branch.
   - Ensure tests pass with `npm test`.
   - Submit a PR using our [PR Template](.github/pull_request_template.md).

## Development Setup

```bash
git clone https://github.com/your-username/ngxsmk-migrate.git
cd ngxsmk-migrate
npm install
npm test
```

## Community Principles

- **Helpful & Technical**: We prioritize quality technical advice and helpful mentorship.
- **Innovative**: Don't be afraid to suggest experimental AI-driven migration patterns.
- **Transparent**: All major roadmap decisions are discussed in the open.

Thank you for helping us modernize the Angular ecosystem!
