# CLI Commands

The `ngxsmk-migrate` CLI provides several commands to help you modernize your Angular app.

## `analyze`

Scans your project and provides an **Angular Upgrade Assistant** report.

```bash
ngxsmk-migrate analyze [path]
```

## `fix`

Runs the **Angular AI Refactor** to automatically update syntax and small logic pieces.

```bash
ngxsmk-migrate fix [path]
```

## `migrate`

The **Standalone Migration Engine** that performs project-wide architecture changes.

```bash
ngxsmk-migrate migrate [path]
```

## `optimize`

The **Angular Signals Migrator** for performance and signal adoption.

```bash
ngxsmk-migrate optimize [path]
```

## `explain`

Explains specific Angular concepts or breaking changes using AI.

```bash
ngxsmk-migrate explain <topic>
```
