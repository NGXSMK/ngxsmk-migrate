# Getting Started with ngxsmk-migrate

`ngxsmk-migrate` is an AI-powered toolkit for modernizing Angular applications.

## Introduction

Modernizing a legacy Angular application can be a daunting task. `ngxsmk-migrate` simplifies this by using AI to analyze, refactor, and migrate your code to the latest Angular standards (Standalone Components, Signals, etc.).

## Prerequisites

- Node.js v18.x or later
- An existing Angular project (v14+)
- A Gemini API Key (for AI features)

## Quick Start

1. **Install the CLI**:
   ```bash
   npm install -g ngxsmk-migrate
   ```
2. **Set your API Key**:
   ```bash
   export GOOGLE_API_KEY=your_key_here
   ```
3. **Analyze your project**:
   ```bash
   ngxsmk-migrate analyze
   ```
